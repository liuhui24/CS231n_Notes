## 1. 为什么研究视觉？

- 视觉是地球上最古老的感官之一，可追溯到 **寒武纪大爆发 (~5.4 亿年前)**
- 人类大脑皮层约 **50% 的神经元**参与视觉处理
- 视觉与神经复杂性的**协同进化**是智能的关键推动力

---

## 2. 生物视觉 vs. 计算机视觉

| | 生物视觉 | 计算机视觉 |
|------|------|------|
| **感知器官** | 眼睛 (视网膜) | 相机 (CMOS/CCD) |
| **核心挑战** | 大脑如何理解视觉信号 | 如何将像素转化为语义理解 |
| **处理方式** | 层级化 (Hubel & Wiesel, 1950s) | 层级化神经网络 |

> **Hubel & Wiesel (1981 诺贝尔奖):** 视觉皮层神经元呈层级结构——先检测边缘 → 再组合为复杂模式 → 直接启发了层级神经网络的设计。

---

## 3. 计算机视觉历史里程碑

| 年代 | 进展 |
|------|------|
| **1963** | Larry Roberts "Block World" — 边缘定义物体形状 |
| **1966** | MIT AI Lab 夏季视觉项目 |
| **1970s** | David Marr 的表征阶段理论: Primal Sketch → 2½D Sketch → 3D Model |
| **1980s** | Fukushima Neocognitron (早期 CNN); 反向传播算法形式化 |
| **1990s** | SIFT 特征, Viola-Jones 人脸检测, 早期工业应用 |
| **1998** | LeCun CNN 用于手写数字识别 (银行/邮政系统) |
| **2009** | **ImageNet** 发布 — 百万级标注图像, 1000 类别 |
| **2012** | **AlexNet** 赢得 ImageNet — **深度学习时代的拐点** |

---

## 4. 2012 年以来的爆发

- 目标检测 → 实例分割 → 图像描述 (Image Captioning)
- 视频理解 → 生成模型 (GAN → Diffusion Models)
- 多模态系统 (Vision + Language)
- 3D 视觉、机器人学习

---

## 5. 推动进步的三大支柱

```
算法 (CNNs → RNNs → Transformers → Diffusion Models)
  ×
数据 (ImageNet, COCO, Web-scale datasets)
  ×
算力 (GPU, TPU, 分布式训练)
```

---

## 6. 2025 课程四大主题

1. **深度学习基础** — 架构、优化、正则化
2. **感知与理解任务** — 分类、检测、分割、视频
3. **大规模训练** — 分布式训练、系统设计
4. **生成式与交互式视觉智能** — 扩散模型、视觉-语言模型、多模态系统

---

## 7. 核心视觉任务一览

| 任务 | 输入 → 输出 |
|------|------------|
| **图像分类** | 图像 → 单个类别标签 |
| **语义分割** | 图像 → 逐像素类别标签 |
| **目标检测** | 图像 → 边界框 + 类别 |
| **实例分割** | 图像 → 逐实例像素掩码 |
| **视频理解** | 视频 → 行为/事件标签 |
