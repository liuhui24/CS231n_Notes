## 1. 机器人学习：第三种范式

不同于监督学习 (静态映射) 和自监督学习 (无标注结构发现)，机器人学习涉及**具身智能体**:

- 感知高维感官输入
- 在物理世界中行动
- 通过学习改善行为
- 运行在**顺序决策循环**中——动作改变环境

---

## 2. MDP 问题框架

每个任务形式化为**马尔可夫决策过程**: $(S, A, P, R, \gamma)$

| 组件 | 描述 |
|------|------|
| **State (S)** | 高维 (RGB/RGB-D) 或低维描述符 |
| **Action (A)** | 离散指令或连续力矩 |
| **Transition (P)** | 环境动力学: $P(S_{t+1} \mid S_t, A_t)$ |
| **Reward (R)** | 标量信号: $R_t = R(S_t, A_t)$ |
| **Discount (γ)** | 即时与未来奖励的权衡 |

**目标:** 找到最优策略 $\pi^*$ 最大化期望累积回报 $J(\pi) = \mathbb{E}_{\tau \sim \pi} [\sum \gamma^t R_t]$

---

## 3. 机器人感知 — 具身、主动、情境化

| 属性 | 含义 |
|------|------|
| **Embodied** | 相机安装在物理身体上；智能体自身动作改变所见 |
| **Active** | 智能体可通过移动选择看哪里——扰动物体、改变视角 |
| **Situated** | 感知与当前任务紧密耦合 |

**模态远超 RGB:** 深度、IMU、力/力矩、触觉、音频。必须在遮挡、噪声、动态环境中融合互补模态。

## 4. 强化学习 (RL)

### RL 与传统监督学习的四大区别

1. **随机性** — 相同 (s, a) 可产生不同 (s', r)
2. **信用分配** — 奖励在很多步之后才到达
3. **不可微** — 无法通过物理世界反向传播
4. **非平稳性** — 策略改进 → 数据分布偏移 (无 i.i.d. 假设)

### 博弈 RL 里程碑

| 时间 | 系统 | 突破 |
|------|------|------|
| 2016.1 | AlphaGo | 模仿+树搜索+RL; 击败李世石 |
| 2017.10 | AlphaGo Zero | 纯自我博弈, 无需人类数据 |
| 2018.12 | AlphaZero | 单一架构覆盖围棋/国际象棋/将棋 |
| 2019.11 | MuZero | 无规则给定时用学习模型规划 |

### 为何 Model-Free RL 在现实世界困难？

- **样本效率低** — 模拟中"40 天自我博弈 3000 年"可行，物理机器人不行
- **安全约束** — 随机探索损坏硬件
- **重置机制** — 物理世界无 `env.reset()`
- **奖励黑客** — 智能体利用设计不良的奖励函数

## 5. 模型学习与基于模型的规划

**核心思想:** 学习前向动力学模型 $f_\theta(s_t, a_t) \rightarrow \hat{s}_{t+1}$，然后**规划**——优化动作序列。

### MPC Pipeline
1. 在 H 步视界上采样/优化动作序列
2. 用 $f_\theta$ 展开预测状态
3. 选择最低成本序列
4. 仅执行第一个动作，然后**重新规划**

### 状态表征

| 表征 | 描述 | 示例 |
|------|------|------|
| **像素级** | 预测图像空间变化 | CDNA Conv+LSTM |
| **关键点** | 稀疏跟踪物体关键点 | KUKA 推 Cheez-It 盒子 |
| **粒子基** | 基于 GNN 的粒子集建模 | RoboCook: 操纵面团/颗粒 |

### RoboCook (CoRL 2023 最佳系统论文)
- 将面团/麦片/米表示为**粒子集**
- 用 **GNN** 学习粒子动力学
- 支持**多阶段层级规划:** 高层工具选择 → 低层轨迹优化
- 从真实数据学习的模型**优于手动调参的物理模拟器**

## 6. 模仿学习

### Behavior Cloning (BC)
简单的监督回归: $\pi_\theta(o_t) \rightarrow a_t$，MSE 最小化

**失败模式 — 协变量偏移 / 级联错误:** 微小偏差 → 分布外状态 → 无恢复样本 → 指数级错误累积

### 解决方案

| 方法 | 思路 |
|------|------|
| **DAgger** | 执行策略 → 在访问状态查询专家 → 迭代重训练 |
| **Inverse RL** | 恢复专家优化的奖励函数 |
| **Implicit BC** | 学习能量函数 $E_\theta(s,a)$; 优化推断 a |
| **Diffusion Policy** (SOTA) | 将动作序列建模为去噪扩散过程，预测 chunk $A_{t:t+H}$ |

### 为什么 Diffusion Policy 是 SOTA？
- **多模态建模** — 自然捕获"向左或向右"而不取平均
- **时序一致性** — 动作 chunk 预测产生平滑连贯运动
- **训练稳定** — 简单 MSE 目标
- **稳定采样** — 在 rollout 内承诺一种行为模式

## 7. 机器人基础模型

### Vision-Language-Action (VLA) 模型
$$(\text{observation}, \text{language instruction}) \rightarrow \text{action}$$

### 进展时间线

| 模型 | 时间 | 关键贡献 |
|------|------|---------|
| **RT-1** | 2022.12 | Transformer-based VLA |
| **RT-2** | 2023.7 | VLM + 机器人控制联合训练 |
| **RT-X** | 2023.10 | 1M episodes, 22 形态, 527 技能, 34 实验室 |
| **OpenVLA** | 2024.6 | 开源: Llama 2 + DINOv2 + SigLIP |
| **π-Zero (π₀)** | 2024.10 | Physical Intelligence; VLM + flow-matching action head |
| **π₀-FAST** | 2025.2 | 开源自回归变体 |

### π-Zero 架构 (Physical Intelligence)
- **VLM 主干:** 语义理解和高层规划
- **Flow-Matching Action Head:** 生成平滑、高频的连续动作轨迹

## 8. 剩余挑战

| 挑战 | 详情 |
|------|------|
| **评估** | 没有"具身 AI 的 ImageNet"; 真实评测噪声大/昂贵; sim-to-real 差距 |
| **基础世界模型** | 动作条件未来预测仍开放; 当前视频模型长期物理一致性丢失 |
| **具身基础模型** | VLM 在几何/物理任务上失败; 需要为具身调优的模型 |
| **数据扩展** | 遥操作数据线性增长; YouTube 人类视频是有前景方向 |
| **终身学习** | 适应新场景、特定人类偏好、自我改进 |

---

## 核心要点

1. **机器人学习本质是顺序的** — 动作改变未来状态，打破监督学习 i.i.d. 假设
2. **感知是具身的** — 机器人必须主动选择感知内容并融合多模态
3. **基于模型的规划** (尤其粒子基 GNN) 实现可形变物体的数据高效操控
4. **Diffusion Policy** 是模仿学习的当前 SOTA
5. **VLA 基础模型** (π₀, RT-2) 正向通用机器人策略收敛，但评估、数据和世界模型仍开放
