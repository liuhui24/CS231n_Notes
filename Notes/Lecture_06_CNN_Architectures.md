## 1. CNN 构建模块回顾

| 模块 | 作用 |
|------|------|
| **卷积层** | 局部特征提取 |
| **池化层** | 空间下采样 (通常 2×2, stride 2) |
| **全连接层** | 全局分类器/回归器 |
| **Batch Norm** | 跨 batch 聚合，逐通道 — 中大 batch |
| **Layer Norm** | 逐样本 — 小 batch, Transformer |

## 2. AlexNet (2012) — 深度学习革命的起点

**ImageNet Top-5 Error: ~15.4%**

| 创新 | 说明 |
|------|------|
| **ReLU** | 替代 sigmoid/tanh，训练更快 |
| **Dropout** | 全连接层正则化 |
| **数据增强** | 随机裁剪 + 水平翻转 + PCA 颜色增强 |
| **GPU 训练** | 2 GPUs 拆分激活图 |

架构: 5 个卷积层 + 3 个全连接层, ~60M 参数

---

## 3. VGG (2014) — 小滤波器、深网络

**VGG16/19, Top-5 Error: ~7.3%**

### 设计哲学
> 只用 3×3 卷积，stride 1, padding 1 + 周期性 2×2 max-pooling

### 为什么三个 3×3 堆叠优于一个 7×7？

| | 3 个 3×3 | 1 个 7×7 |
|------|---------|---------|
| **感受野** | 3→5→7 (等效) | 7 |
| **非线性层** | 3 | 1 |
| **参数量** | 3×3²×C² = 27C² | 7²×C² = 49C² |

---

## 4. ResNet (2015) — 跳跃连接革命

**ResNet-152, Top-5 Error: ~3.6%**

### 核心问题：退化
普通网络加深 → 训练误差和测试误差都升高。这不是过拟合，是**优化困难**。

### 解决方案：残差学习

$$\text{Output} = F(x) + x$$

残差块: `Conv → BN → ReLU → Conv → BN → (+ skip) → ReLU`

### 为什么有效？
1. 恒等路径提供梯度"高速公路"
2. L2 正则化驱动权重→0 → 零权重残差块 = 恒等映射
3. 梯度通过跳跃连接直接回传

### ResNet 分阶段设计
- 每个阶段: 空间下采样 + 滤波器数翻倍
- ResNet-50+: Bottleneck 块 (1×1→3×3→1×1)

### 训练配方
| 超参数 | 设置 |
|--------|------|
| 归一化 | 每个 CONV 后加 BN |
| 初始化 | He (Kaiming) 初始化 |
| 优化器 | SGD + Momentum (0.9) |
| 学习率 | 0.1, 停滞时 ÷10 |
| Weight Decay | 1e-5 |
| Dropout | **不使用** (BN 已足够) |

## 5. 训练实践

### 数据增强
- 水平翻转、随机缩放裁剪、颜色抖动、Cutout
- **测试时增强 (TTA):** 多裁剪/尺度平均 (~1-2% 提升)

### 迁移学习
- 小数据集: 冻结预训练特征，只训练新分类器
- 大数据集: 微调所有层

### 超参数调优
1. 先在小子集上过拟合 → 验证实现
2. 学习率粗搜索
3. 随机搜索 (非网格)
4. 监控 train/val loss

## 6. 架构演进总结

| 架构 | 年份 | 关键创新 | Top-5 Error |
|------|------|---------|------------|
| **AlexNet** | 2012 | ReLU, Dropout, GPU | ~15.4% |
| **VGG** | 2014 | 统一 3×3, 小滤波器堆叠 | ~7.3% |
| **GoogLeNet** | 2014 | Inception 模块, 1×1 瓶颈 | ~6.7% |
| **ResNet** | 2015 | 跳跃/残差连接, 152 层 | ~3.6% |
| **ConvNeXt** | 2022 | 用 Transformer 思路现代化 ResNet | — |

---

## 核心要点

1. 小滤波器堆叠 > 单个大滤波器 — 更多非线性 + 更少参数
2. 残差连接是最重要的架构创新 — 梯度高速公路
3. 归一化 + 好的初始化 + 增强是训练必需品
4. 架构从手工走向系统化: AlexNet → VGG → ResNet → ConvNeXt
