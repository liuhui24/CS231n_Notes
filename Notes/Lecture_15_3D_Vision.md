## 1. 3D 视觉概览

四大核心应用:
- **生成** — 合成新 3D 形状
- **重建** — 从图像/扫描恢复 3D
- **渲染** — 从 3D 模型生成 2D 视图
- **动画** — 变形和关节化 3D 几何

---

## 2. 3D 表征: 显式 vs 隐式

### 显式表征
直接列出几何基元。**易采样，难查询内外。**

#### a) 点云 (Point Clouds)
- 无序 3D 点集: $\mathcal{P} = \{P_i\}_{i=1}^N$
- **优点:** 传感器原生 (LiDAR, RGB-D), 支持任意拓扑
- **缺点:** 无连接/拓扑，噪声，采样不均，难以平滑渲染
- **关键属性:** 无结构，排列不变，密度不均

#### b) 多边形网格 (Meshes)
- 顶点 + 面: $\mathcal{M} = (V, F)$
- **优点:** 连接性支持细分/简化/重网格化，高质量渲染
- **缺点:** 不规则数据，神经网络集成难

#### c) 参数化表征
- 从 2D 参数域到 3D 的映射: $\mathbf{x}(u,v) = f(u,v)$
- Bézier 曲面片、B-spline/NURBS
- **优点:** 紧凑，精确求值，CAD 友好
- **缺点:** 复杂有机拓扑需要很多曲面片

### 隐式表征
将几何编码为函数 $f(x)$ — 表面是零水平集 $\{x \mid f(x)=0\}$。

#### a) 有符号距离函数 (SDF)
- $f(\mathbf{x})$ = 到表面的有符号距离
- 梯度给出法线: $\mathbf{n} = \nabla f / \|\nabla f\|$
- Eikonal 约束: $\|\nabla f\| \approx 1$

#### b) 体素 (Voxels)
- 在均匀 3D 网格上离散化隐式场
- **优点:** 直接兼容 3D CNN
- **缺点:** O(N³) 内存，大多为空

## 4. 深度学习方法

### A. Multi-View CNN (MVCNN)
- 从 K 个视点渲染 3D 物体 → 共享 2D CNN → 聚合
- **优点:** 复用 ImageNet 预训练
- **缺点:** 丢失几何信息

### B. PointNet 家族

**PointNet:**
$$f(\mathcal{P}) = \gamma(\text{POOL}(\phi(p_1), \dots, \phi(p_N)))$$

- 逐点 MLP → 对称聚合 (max-pooling) → 全局描述符
- 保证排列不变性
- **局限:** 仅全局聚合，无局部结构建模

**PointNet++:**
- 层级化: 最远点采样 (FPS) → 局部分组 (kNN/radius) → 逐 patch 局部 PointNet
- 类比 CNN 层级结构

### 点集损失函数

| 损失 | 描述 |
|------|------|
| **Chamfer Distance** | 双向最近邻匹配 — 简单可微，但允许点坍缩 |
| **Earth Mover's Distance (EMD)** | 最优传输一对一匹配 — 更严格，几何质量更好 |

### C. AtlasNet
- 学习从 2D 参数 patch 到 3D 表面的神经映射
- 多 patch 覆盖复杂形状
- 产生平滑连续表面

### D. Deep Implicit Functions
- 神经网络映射 $(x,y,z) \rightarrow$ occupancy / SDF
- **优势:** 任意分辨率求值，自然组合

### E. NeRF (Neural Radiance Fields) — 转折点
- 网络以**位置** $(x,y,z)$ 和**视角** $(\theta,\phi)$ 为条件
- 输出: **密度** $\sigma$ (几何) + **颜色** (外观)
- **可微体积渲染:** 沿相机射线积分密度 × 颜色
- **关键洞察:** 可仅从 2D 图像训练 — 逐射线颜色损失反向传播通过渲染过程

### F. Gaussian Splatting
- 解决 NeRF 效率问题
- 将场景参数化为**各向异性 3D 高斯** ("splats")
- 高斯放置在几何和外观集中处 — 避免采样空区域
- 比 NeRF 快得多

## 5. 显式 vs 隐式 权衡总结

| 标准 | 显式 (点云/Mesh) | 隐式 (SDF/NeRF) |
|------|-----------------|----------------|
| 表面采样 | 容易 | 困难 |
| 内外查询 | 困难 | 容易 (评估 f(x)) |
| 渲染 | Mesh: 优秀; 点云: 需 splatting | 可微体积渲染 |
| 神经网络集成 | PointNet 等专用架构 | MLP 评估，自然利于优化 |
| 拓扑变化 | 困难 | 自然 (水平集合并/分裂) |
| 存储 | Mesh: 紧凑; 点云: 可变 | 隐式: 紧凑网络; 体素: O(N³) |

---

## 核心要点

1. 3D 表征分为**显式** (点云/Mesh) 和**隐式** (SDF/NeRF)，各有不同权衡
2. **PointNet/PointNet++** 是点云处理的基石架构
3. **NeRF** 是分水岭 — 可微渲染使从 2D 图像学习 3D 成为可能
4. **Gaussian Splatting** 提供 NeRF 的高效替代
5. 3D 数据稀缺 → 利用 2D 先验和多视图监督是主流方向
