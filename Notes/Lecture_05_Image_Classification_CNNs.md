#  Lecture 5: Image Classification with CNNs

---

## 1. 从全连接到卷积

### 全连接层的问题

- 每个输出是所有输入像素的加权和 → **全局模板匹配**
- 破坏图像的 2D 空间结构；参数量巨大

### 卷积层的解决方案
学习**小尺寸、空间局部化的滤波器 (kernel)**，在图像上滑动计算。

---

## 2. 卷积层机制

| 参数 | 描述 |
|------|------|
| **滤波器形状** | `(C_out, C_in, K_h, K_w)` |
| **操作** | 每个滤波器在空间上滑动，每个位置计算与局部 patch 的点积 |
| **输出** | `(N, C_out, H', W')` |

### 三大归纳偏置
| 偏置 | 含义 |
|------|------|
| **局部性** | 滤波器只看小空间邻域 |
| **权重共享** | 同一滤波器在所有空间位置复用 |
| **平移等变性** | 平移输入 → 输出相应平移 |

---

## 3. 空间维度计算

$$W' = \frac{W - K + 2P}{S} + 1$$

- **Same Padding:** 选奇数核 (3×3, 5×5), $P = (K-1)/2$ → 输出尺寸 = 输入尺寸

---

## 4. 参数量与计算量

| 指标 | 公式 |
|------|------|
| **参数量** | $C_{out} \times C_{in} \times K_h \times K_w + C_{out}$ |
| **FLOPs** | $N \times C_{out} \times H' \times W' \times C_{in} \times K_h \times K_w$ |

---

## 5. 池化层

| 类型 | 操作 |
|------|------|
| **Max Pooling** | 取每个窗口的局部最大值 (非线性) |
| **Average Pooling** | 取每个窗口的局部平均值 (线性) |

**典型配置:** 2×2 核, stride 2 → 空间尺寸减半, 通道数不变

---

## 6. 学习到的滤波器层级

| 深度 | 检测内容 |
|------|---------|
| **Layer 1** | 定向边缘、颜色对立斑点 (Gabor 类) |
| **中层** | 纹理、角点、简单模式 |
| **深层** | 物体部件: 眼睛、车轮、文字模式、语义主题 |

---

## 7. 典型 CNN 架构模式

```
INPUT → [[CONV → ReLU]*N → POOL?]*M → [FC → ReLU]*K → FC → SOFTMAX
```

- **Body:** 卷积 + 非线性 + 可选池化 → 层级特征提取
- **Head:** 小全连接分类器 → 特征映射到类别得分
- **所有参数端到端学习**

---

## 8. 卷积变体

| 维度 | 应用 |
|------|------|
| **1D 卷积** | 序列、时序、文本 |
| **2D 卷积** | 图像 (标准) |
| **3D 卷积** | 体积数据、视频 (时空) |
| **1×1 卷积** | 通道维度降维/扩展 (逐点卷积) |

## 9. 历史脉络

```
1998 LeNet → 2012 AlexNet → 2012-2020 CNN 主导 → 2020- ViT 和混合架构崛起
```

---

## 核心要点

1. CNN **保留空间结构**，利用局部性 + 权重共享作为归纳偏置
2. 架构定义可学习函数族——人设计算子序列，数据决定滤波器
3. 非线性是必需的——否则深度不增加表征能力
4. 学习到的滤波器形成层级: 边缘 → 纹理 → 部件 → 物体
5. 虽然 Transformer 崛起，CNN 原语仍是现代 CV 的基础
