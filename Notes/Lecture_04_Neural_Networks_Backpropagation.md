## 1. 从线性分类器到神经网络

```
单层:  s = Wx + b          (D 输入 → C 输出)
双层:  f = W₂ · max(0, W₁x + b₁) + b₂
```

### 为什么必须要有非线性？

> 没有非线性激活函数，多层线性层塌缩为单个线性映射。无论多少层，仍是线性的。

---

## 2. 激活函数

| 函数 | 公式 | 优点 | 缺点 |
|------|------|------|------|
| **ReLU** | max(0, x) | 计算高效，缓解梯度消失 | Dead neurons |
| **Leaky ReLU** | max(αx, x) | 不"死亡" | 需调 α |
| **ELU** | x if x>0 else α(eˣ−1) | 零中心，鲁棒 | exp() 昂贵 |
| **Sigmoid** | 1/(1+e⁻ˣ) | 压缩到 [0,1] | 梯度消失，非零中心 |
| **Tanh** | (eˣ−e⁻ˣ)/(eˣ+e⁻ˣ) | 零中心 [-1,1] | 梯度消失 |
| **Swish** | x·σ(x) | 现代架构中表现好 | — |

**默认选择:** ReLU (尤其 CNN)，但具体选择是经验性的。

---

## 3. 计算图与反向传播

### 核心原理

> **下游梯度 = 上游梯度 × 局部梯度**

### 示例: `f = (x + y) * z` (x=-2, y=5, z=-4)

| 步骤 | 前向 | 反向 |
|------|------|------|
| q = x + y | q = 3 | ∂f/∂x = -4, ∂f/∂y = -4 |
| f = q * z | f = -12 | ∂f/∂z = 3, ∂f/∂q = -4 |

---

## 4. 常见梯度门模式

| 门 | 前向 | 反向规则 |
|------|------|---------|
| **Add** | x + y | 梯度不变地分发给所有输入 |
| **Multiply** | x × y | 交换变量: ∂(xy)/∂x = y |
| **Copy** | → 多个 | 求和所有消费者的梯度 |
| **Max / ReLU** | max(x, y) | 只路由给获胜输入 |
| **Sigmoid** | σ(x) | σ(x)(1−σ(x)) — 仅用前向值即可 |

---

## 5. 矩阵反向传播

对于 `Y = X·W` (X: [N,D], W: [D,M]):

| 梯度 | 公式 |
|------|------|
| ∂L/∂X | ∂L/∂Y · Wᵀ |
| ∂L/∂W | Xᵀ · ∂L/∂Y |

> **关键:** 永远不要显式构造完整 Jacobian。用维度分析推导高效反向传播。

---

## 6. 实践: 两层网络实现步骤

1. 定义输入/输出维度和隐藏层大小
2. 初始化权重 (He/Kaiming 初始化)
3. 前向传播: 计算中间表示 → 得分 → 损失
4. 反向传播: 计算所有可训练参数的梯度
5. 梯度检查: 用数值梯度验证
6. 用解析梯度训练

---

## 7. 模型容量与过拟合

| 更多隐藏单元 | → 更大容量 → 更复杂的决策边界 |
|-------------|-------------------------------|
| **容量过大** | → 过拟合 |
| **解决方案** | 正则化 (L2, Dropout)，而非减少神经元 |

---

## 核心要点

1. **反向传播** = 链式法则在计算图上的系统应用
2. 每个可微操作都是具有局部梯度规则的门
3. 模块化 API (`forward()` + `backward()`) 是所有 DL 框架的设计模式
4. 非线性是必需的——否则深度网络退化为线性模型
5. Add 门分发、Multiply 门交换、Max 门路由
