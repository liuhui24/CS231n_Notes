# Lecture 10: Video Understanding

---

## 1. 为什么研究视频？

- 视频理解是静态图像任务之后的**自然下一步**
- 视频引入**时间维度**: 不仅要理解画面中*有什么*，还要理解*正在发生什么*
- 目标是构建**多感官机器智能** — 视觉 + 音频 + 触觉等

### 视频分类 vs 图像分类

| | 图像分类 | 视频分类 |
|------|---------|---------|
| **关注点** | 静态物体外观 | **活动和时序动态** |
| **数据形式** | 3D 张量 (C×H×W) | **4D 张量 (C×T×H×W)** |

---

## 2. 视频作为数据结构

```
视频张量: C × T × H × W
  C = 通道 (3 for RGB)
  T = 帧数 (时间)
  H, W = 高, 宽 (空间)
```

### 数据规模挑战

- 高空间 + 时间分辨率 → 每分钟数 GB 原始视频
- GPU 内存和计算严格受限

### 预处理策略

| 策略 | 描述 |
|------|------|
| **空间下采样** | 降低每帧分辨率 |
| **时间子采样** | 降低帧率 (FPS) |
| **Clip 训练** | 用短固定长度 clip (16/32 帧) 滑动窗口训练 |

### 推理协议

对多个采样 clip 分别分类 → **聚合** (平均/投票) → 视频级标签

---

## 3. 时序融合策略

### 融合谱系

```
单帧 Baseline → Late Fusion → Early Fusion → Slow Fusion (3D CNN)
```

### 3.1 单帧 Baseline
- 逐帧独立跑 2D CNN → 聚合预测
- **惊人地强** — 常常难以被超越

### 3.2 Late Fusion
- 逐帧提取 2D CNN 特征 → 拼接或池化 → FC 分类
- 拼接: 参数爆炸; 池化: 丢失时序顺序

### 3.3 Early Fusion
- 将所有帧沿通道轴拼接 (T=16 → 3×16=48 通道)
- 第一层就混合时间信息
- **局限:** 试图在单层捕获所有运动，缺乏层级时序建模

### 3.4 Slow Fusion (3D CNN) — 最佳折中
- 在网络深度上逐步融合时空信息
- 使用 **3D 卷积**和 **3D 池化**
- 层级化时空特征学习

---

## 4. 3D 卷积

### 操作
- 输入: `C × T × H × W`
- 核: `kT × kH × kW` (跨时间和空间滑动)
- 提供**时序平移不变性** (2D 核跨整个时间轴则缺乏)

### 可视化
学习到的 3D 滤波器可被可视化为**短视频片段**，展示颜色过渡、运动边缘、方向运动模式

### 计算代价
- 将 2D 架构膨胀为 3D → **约 10× FLOPs**
- 需要高效架构设计

---

## 5. 关键架构

### C3D
- 早期有影响力的 3D CNN
- 在 Sports1M (数百体育类别) 上训练
- 作为预训练特征提取器发布

### Two-Stream Networks
- **核心洞察:** 空间外观和时间运动是根本不同的信号
- **RGB Stream:** 静态外观
- **Optical Flow Stream:** 运动信息 (水平+垂直光流)
- **光流:** 相邻帧之间的逐像素 2D 运动向量
- 光流 stream 单独使用就出奇强大

### I3D (Inflated 3D)
- 将 2D CNN 核**膨胀**为 3D (`kT × kH × kW`)
- 复用成功的 2D 架构 (如 Inception)
- **关键技巧:** 从预训练 2D 权重初始化 3D — 复制并沿时间维度缩放

### CNN + RNN/LSTM
- CNN 提取 clip 级特征 → RNN/LSTM 建模更长程时序依赖
- 挑战: 梯度消失、训练稳定性、并行化限制

### Non-Local / Self-Attention
- 将 Self-Attention 扩展到视频
- 计算 `C×T×H×W` 特征图上的 Q/K/V (通过 1×1×1 卷积)
- 任意时空位置间的 Attention

### 现代 Transformer 视频模型
- 分解时空 Attention (Factorized Space-Time)
- MAE 风格 (Masked Autoencoder) 视频预训练
- Kinetics-400 上大幅提升

---

## 6. 实验发现

| 方法 | 性能 |
|------|------|
| 单帧 Baseline | 惊人地强 |
| Early Fusion | 可能更差 |
| Late Fusion | 略优于 baseline |
| 3D CNN (C3D 等) | 适度提升，依赖算力 |
| Two-Stream (RGB + Flow) | 强，Flow stream 信息量很大 |
| 现代 Transformer + 预训练 | 大规模下大幅提升 |

---

## 7. 超越分类：更多视频任务

- **时序动作定位:** 在长视频中识别动作发生的**时间区间**
- **时空检测:** 在**空间和时间**上联合定位动作
- **视频可视化:** 基于梯度的输入 RGB/Flow 优化以最大化类别得分

---

## 8. 多模态与音视频学习

### 视觉引导的音频源分离
- 用视觉线索 (嘴唇运动、乐器运动) 分离混合音频
- 应用: 噪声环境中的语音分离、乐器隔离

### 自我中心视频 (Egocentric)
- 来自可穿戴设备的第一人称多模态流
- 任务: 说话者-听者识别、社交交互理解

## 9. 未来：视频基础模型

- Tokenize 视觉+音频内容 → 映射到 LLM 嵌入空间
- 实现**可 prompt 的视频理解**和描述
- 连接视频表征学习与自然语言界面

---

## 核心要点

1. **视频 = 图像 + 时间** — 时间维度引入新的建模挑战
2. **融合谱系:** Late → Slow (3D) → Early，各有不同的参数/建模权衡
3. **3D 卷积**提供 2D 核缺乏的时序平移不变性
4. **Two-Stream** (RGB + 光流) 显式分离外观和运动
5. **I3D 膨胀**是实用技巧: 复用预训练 2D 权重初始化 3D 模型
6. 现代 Transformer + 自监督预训练驱动 SOTA
7. 领域正向**视频基础模型**方向发展，连接视觉与 LLM
