## 1. 图像分类问题定义

**输入:** 图像 → 3D 张量 `H × W × C` (RGB: 3 通道, 像素值 0-255)
**输出:** 一个类别标签 (如 {猫, 狗, 卡车, 飞机})

**核心困难 —— "语义鸿沟":** 原始像素值与人类语义理解之间存在巨大差距

---

## 2. 图像分类的七大挑战

| 挑战 | 说明 |
|------|------|
| **视角变化** | 同一物体不同角度拍摄差异巨大 |
| **光照变化** | 不同光照下像素值天差地别 |
| **尺度变化** | 物体在图像中大小不一 |
| **遮挡** | 物体可能只露出一小部分 |
| **形变** | 非刚体物体可以变形 |
| **背景干扰** | 物体与背景融为一体 |
| **类内差异** | 同一类别内外观多样性 (如各种狗) |

## 3. 数据驱动方法

> 不手工设计规则，而是从数据中学习

**三步流程:** 1. 收集带标签的图像数据集 → 2. 训练模型拟合参数 → 3. 在测试集上评估

## 4. k-近邻 (k-NN)

### 训练: O(1)
"训练"即记住所有训练数据 (非参数方法)

### 预测: O(N)
对每个测试图像，计算与所有训练图像的距离

**L1 (曼哈顿距离):** $d_1(I_1, I_2) = \sum_p |I_1^p - I_2^p|$

**L2 (欧氏距离):** $d_2(I_1, I_2) = \sqrt{\sum_p (I_1^p - I_2^p)^2}$

| k 值 | 效果 |
|------|------|
| k=1 | 完美记忆训练数据，易过拟合 |
| k 较大 | 更平滑边界，更鲁棒 |
| CIFAR-10 最佳 k≈7 | 验证准确率 ~28% |

## 5. 超参数调优

> **黄金法则:** 测试集只在最终评估时使用**一次**
> ❌ 在测试集上调参 → 信息泄露
> ✅ 划分 train + val，在 val 上调参
> ✅ K-fold 交叉验证 (小数据集)

## 6. 线性分类器

### 得分函数

$$f(x_i, W, b) = W x_i + b$$

- **W:** 权重矩阵 `[K × D]`，每行是一个类别的分类器模板
- **b:** 偏置向量 `[K × 1]`

### 两种几何解释
- **模板匹配:** W 的每行是一个"类别模板"，点积衡量匹配程度
- **超平面划分:** 高维空间中用超平面分隔各类别

### 局限
- 无法解决 XOR 类问题；无法处理多模态分布
- CIFAR-10 模板出现"双头马"——左右朝向被平均成一个模板

---

## 7. 损失函数

### Multiclass SVM Loss (合页损失)

$$L_i = \sum_{j \neq y_i} \max(0, \; s_j - s_{y_i} + \Delta)$$

### Softmax + 交叉熵

$$L_i = -\log\left(\frac{e^{s_{y_i}}}{\sum_j e^{s_j}}\right)$$

### 正则化

$$L(W) = \frac{1}{N}\sum_i L_i + \lambda R(W)$$

- **L2:** $R(W) = \sum W^2$ — 偏好小且分散的权重
- **L1:** $R(W) = \sum |W|$ — 鼓励稀疏解

---

## 8. k-NN vs 线性分类器

| 方面 | k-NN | 线性分类器 |
|------|------|-----------|
| **类型** | 非参数 | 参数化 |
| **训练** | O(1) 记忆数据 | 学习 W, b (优化) |
| **预测** | O(N) 比较所有数据 | O(1) 一次矩阵乘法 |
| **CIFAR-10** | ~28% | ~35-40% |

> 线性分类器是神经网络的**基本构建块**。堆叠线性层 + 非线性激活函数 → 复杂非线性决策边界。
